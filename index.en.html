twicli

Bookmark this on Delicious

twicli is a browser-based Twitter client.

  • No need for installation. All you need is a Web browser.
  • Automatically fetches timeline, and show with cool animation.
  • Use cross domain JavaScript to access Twitter directly (no server needed), so your password is sent to Twitter only.

Useful when you are in an Internet cafe, doing something else while viewing Twitter timeline.
Works on Firefox, Safari, Opera, IE8 at the moment, as well as Safari on iPhone/iPod touch
Works on IE7 too, but some parts aren't displayed properly.

Usage

Launch

Just click the link above, or run it as a bookmarklet.
If you get asked for authorization, check if the origin is really "twitter.com", then enter your Twitter account/password.

Be careful not to hit API request limit (150 requests per hour) by clicking too much ^^;

How to Use

Customize

User Stylesheet

You can add your own CSS in "user stylesheet" section of the "+" tab. For example;
  • to hide user icons : .uicon { display: none; }
  • wider lines : .status { line-height: 1.5; }

Plugins

You can register plugins by setting them at the "Preferences" in the "+" tab.
Currently, there are plugins below:
  • regexp.js : Add a tab that extracts timeline that has specific keywords, ID, etc. Detail(in Japanese)
  • lists.js : Add a tab that shows lists you following. Specify the name of lists to make tab in "+" tab.
  • outputz.js : Count how many letters you tweet, and send it to Outputz.
  • search.js : Add keyword search by Twitter Search to the "+" tab.
  • search2.js : Add keyword search by twitter search(yats) to the "+" tab.
  • favotter.js : Indicate latest popular favorited tweets like [fav:3]. Thanks to Favotter.
  • followers.js : Colorize tweets of users that you don't follow in blue. To use this, you need to fetch the list of followers by clicking the "Renew" button at the "Color followers" section.
  • sound.js : Play sound when updating timeline/reply. Not registedred as default.
  • ssl.js : Use SSL to get timeline and to tweet. Not registedred as default.
  • translate.js : Choose "translate" from â–¼, then the tweet is translated into the language specified in "+" tab > Translate. (Specify "en" for English.)
  • thumbnail.js : Add thumbnails to the URL of some photo sharing services. Currently supports twitpic, movapic, HatenaFotolife, Tumblr, Yfrog, and Flickr.
  • resolve_url.js : Replace short URLs with the originals in the timeline. (created by @edvakf)
  • shorten_url.js : Typing a URL in the text area followed by three semicolons (;;;) makes a shortened URL. (created by @edvakf)

FAQ

  • How to hide tweet which contains specified kerwords?
    => Write "-::keyword:1" into "Pickup Pattern" in the "+" tab.
  • My tweets are sometimes lost ...
    => Twitter may be too crowded. Try writing just "r" and hit enter to re-post your previous tweet.
  • Appearance of this page is poor ...
    => Because of displaying many posts at once, and my laziness .... There are some guys publishing stylesheets for twicli. Your proposal is also appreciated. :-)
  • Too much API usage when I tweet frequently ...
    => Disable "Update after post" in "Preferences" in "+" tab to suppress automatic update after tweet is posted.
